# Logaritmi
| Proprietà | Funzione |
| --------- | :------: |
| **Rapporto** | $log_c{\frac{a}{b}} = log_c{a} - log_c{b}$ |

# Sommatorie
| Proprietà | Funzione |
| --------- | :------: |
| **Associativa** | $\sum_{i = n}^{m}{f(i)} \pm \sum_{i = n}^{m}{g(i)} = \sum_{i = n}^{m}{(f(i) \pm g(i))}$ |
| **Dissociativa** | $\sum_{i = n}^{m}{(f(i) \pm g(i))} = \sum_{i = n}^{m}{f(i)} \pm \sum_{i = n}^{m}{g(i)}$ |
| **Progressione<br>aritmetica**| $\sum_{i = 0}^{n - 1}{i} = \frac{n(n - 1)}{2}$ |
| **Progressione<br>geometrica**| $\sum_{i = 0}^{n}{x^i} = \frac{x^{n + 1} - 1}{x - 1}$ |
| **Serie finita $ix^i$**| $\sum_{i = 0}^{n}{ix^i} = x\frac{1 - x^n}{(1 - x)^2} - \frac{nx^{n + 1}}{1 - x}$ |
